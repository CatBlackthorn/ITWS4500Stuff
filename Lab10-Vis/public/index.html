Welcome to TweetRead


All of the fill in the boxes on the next page have default contents. For search, there are two boxes, one listing RPI and the other listing a number. Please change RPI to whatever you wish to search in tweets. The number determines how many are retrieved, note. There can be less than the number you asked for due to Twitter not finding that number of tweets in the search time. There is also a box containing 'lab8', this is the file name that you want the tweets to be written to.

There are several buttons on the following page, below are descriptions of them.


Search
Adds the search query results to the database
Display
Prints all searches out onto the page with the date/time you searched it
Clear Table
Empties out the database and all searches completely
Export
Saves a file in specified format on the server
